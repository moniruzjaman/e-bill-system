# SOP: Generating Professional Subsidy Summaries

## Goal
Generate a structured report for **Agriculture Officers** summarizing the subsidy distribution across farmers and dealer transactions.

## Inputs
- Transaction data from `.tmp/demo_data.json`

## Execution Steps
1. Run the Python execution script `execution/process_subsidies.py`.
2. Ensure the output is formatted as a clean markdown table.
3. Highlight any anomalies (e.g., zero subsidy for commercial dealers).

## Outcome
A clear, audit-ready summary for the regional officer.
